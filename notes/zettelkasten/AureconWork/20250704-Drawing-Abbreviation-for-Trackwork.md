# Drawing Abbreviation for Trackwork

## Drawing Abbreviation for trackwork that we should know.
- **BS** : Buffer stop
- **CH** : Chainage
- **CL** : Center Line
- **MIN** : Minimum
- **PC** : Beginning of Curve
- **PI** : Point of Intersection
- **POB** : Point of Beginning
- **POE** : Point of End
- **POT** : Point of Tangent
- PT : Ending of Curve
